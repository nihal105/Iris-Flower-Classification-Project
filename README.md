# Iris-Flower-Classification-Project

![Iris Dataset](iris.png)

## 📌 Overview

This project focuses on building a **Machine Learning model** to classify iris flowers into three species:

*  Setosa
*  Versicolor
*  Virginica

The classification is based on four key features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

---

##  Objective

The main goal is to develop a model that can **accurately predict the species of an iris flower** using supervised learning techniques.

---

##  Dataset

* Source: Built-in dataset from **Seaborn / Scikit-learn**
* Total Records: **150**
* Features: **4 numerical features**
* Target: **Species (3 classes)**

---

##  Technologies Used

* Python 
* Pandas

##  Project Workflow

1. **Data Collection**

   * Loaded dataset using Seaborn

2. **Data Preprocessing**

   * Checked missing values
   * Removed duplicates
   * Verified data types

3. **Exploratory Data Analysis (EDA)**

   * Statistical summary
   * Boxplot for outlier detection

4. **Feature Engineering**

   * Feature selection
   * Data scaling using `StandardScaler`

5. **Model Building**

   * Logistic Regression model

6. **Model Evaluation**

   * Accuracy Score
   * Confusion Matrix
   * Classification Report

---

##  Model Performance

The Logistic Regression model performs well on the dataset with high accuracy due to the simplicity and quality of the Iris dataset.
